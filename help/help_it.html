JStruct user help

Il programma JStruct permette l'editing interattivo di programmi Java usando blocchi strutturati (NSD: Nassi-Shneiderman Diagrams) per implementare gli algoritmi.
Il programma offre editing grafico interattivo con funzioni copia, sposta, zoom, espandi etc. commenti per ogni blocco strutturato ed inline nel codice, import ed export di file source java, export di immagini in vari formati raster e vettoriali, elevata interattività: tooltip e menu contestuali, funzionalità undo e redo.
JStruct può essere usato nei modi seguenti:
  • Sviluppo: creando da zero un nuovo programma java, si utilizza l'interfaccia grafica per aggiungere nuovi elementi strutturati, commenti etc...
  • Reverse engineering: evidenziando la struttura di Classi e Metodi si facilita la comprensione del codice. Inoltre si possono usare i colori, aggiungere commenti e cambiare nomi alle variabili.
  • Documentazione: esportando un'immagine si ha completo controllo sui font, colori, dettagli.
  • Tool add-on: aggiungendo JStruct ai tool di un IDE (e.g. JCreator) si aggiunge un editor grafico strutturato all'ambiente integrato.
  • Pretty-printer: un passaggio di un source java in JStruct (import + export) produce un output formattato standard.

Controllo comportamento

      Su base singolo file, flag inseriti nel primo commento:
       //@JStructOptions: ADDJAVADOCPUBLIC, PROGRAMDOCONTOP, SETSOURCEVERSION
  • PROGRAMDOCONTOP se presente i commenti 'inline' ( usano '//' )  sono posti sopra al codice, altrimenti sono posti a destra del codice.
  • ADDJAVADOCEXTRA se presente aggiunge javadoc commenti a metodi e classi
  • ADDJAVADOCPUBLIC limita i commenti a classi e metodi pubblici (esclusi gli @override).
  • SETSOURCEVERSION aggiunge/aggiorna un contatare di "buid" proggressivo.
  • ADDJAVADOCIMAGE  aggiunge nel commento un link ad una immagine esportata NSD (in javadoc/resources).
